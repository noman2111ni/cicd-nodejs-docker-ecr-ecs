# CI/CD Pipeline - Node.js + Docker + AWS ECS

## Overview
Automated CI/CD pipeline using GitHub Actions that builds, 
tests and deploys Node.js application to AWS ECS.

## Pipeline Flow
Build → Test → Deploy

## Tech Stack
- Node.js
- Docker
- GitHub Actions
- AWS ECR
- AWS ECS

## Pipeline Steps
1. Code push to main branch
2. Build job runs
3. Test job runs
4. Deploy to AWS ECS
